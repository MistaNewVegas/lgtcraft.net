How to Install the "Create Astral Space Beyond" Modpack

  1. Download and install the CurseForge app.
  2. Open CurseForge and select the "Minecraft" tab.
  3. Click "Browse Modpacks" and search for Create Astral Space Beyond.
  4. Select the modpack and click "Install".
  5. After installation, click "Play" to launch Minecraft with the modpack.
  6. In Minecraft, go to Multiplayer > Add Server, enter play.lgtcraft.net, then click "Done".
  7. Select the server and click "Join Server"!
